# VS_Code_Shortcuts

### 라인

- Ctrl + [] : 라인 들여쓰기
- Ctrl + Alt + Down(Up) : 커서를 아래(위) 추가
- Shift + Alt + Down(Up) : 아래(위)에 행 복사 추가

### 단어

- Ctrl + F2 : 현재 단어의 모든 항목 선택
- Ctrl + D : 커서가 위치한 같은 단어 반복 선택

### 주석

- Ctrl + / : 한 줄 주석
- Shift + Alt + a : 전체 주석

### 에디터/윈도우 관리
- Ctrl+1	첫번째 편집기 그룹에 초점 맞추기
